YOUTUBE

https://console.developers.google.com/apis/library

- Search for Youtube and choose "Youtube Data API v3"
- Then it'll opne the library page
- Click in "Enable"
- It will ask you to create a new Project
- Chose a name for the project and "Create"
- Now that your project has been created you'll be able to enable the library

- Now you just need to "Create Credentials", by clicking in the top right button
- In the second dropdown choose the option "Other UI"
- And check the Public Data

- Click in the bottom button and it will give you a API Key
